Base shell comand
-----
ls - list directory
pwd - show current path
cd - change directory
mkdir - create empty dir (or tree of directorys with flag -r exaple **mkdir /lol/kek/cheburek)
rm - remove file (use rm -r to remove non empty directory)
rmdir - remove directory
cat - show text file
